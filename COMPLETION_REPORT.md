## 💰 费用计算模块

### 实现功能
```python
calculate_buy_fee(amount)       # 买入费用
calculate_sell_fee(amount)      # 卖出费用
```

### 计算规则

**买入费用**:
- 每1万元收3元
- 不满1万仍收3元
- 示例：15,000元 → 6元（3+3）

**卖出费用**:
- 印花税：每1万5元
- 交易费：每1万3元
- 不满1万仍收全额
- 示例：15,000元 → 16元（8+8）

### 在交易中的应用
```python
# 买入时
total_cost = buy_cost + calculate_buy_fee(buy_cost)

# 卖出时  
net_proceeds = proceeds - calculate_sell_fee(proceeds)
```
