# Trade_real - 实盘交易模块

## 项目结构

```
Trade_real/
├── config.py              # 配置文件（QMT路径、账户ID等）
├── ma5_strategy.py        # 5日均线策略（核心交易逻辑）
├── run_strategy.py        # 策略启动脚本
└── README.md              # 本文件
```

## 文件说明

### config.py
配置文件，包含：
- QMT安装路径
- 账户ID
- 每只股票的资金分配
- 选股参数
- 卖出参数
- 交易参数

### ma5_strategy.py
核心策略文件，包含：
- `StrategyCallback`: 事件回调类，处理订单、成交等事件
- `MA5Strategy`: 5日均线策略类，实现了：
  - 账户管理（连接、查询资产等）
  - 数据获取（行情数据、历史数据）
  - 技术指标计算（移动平均线）
  - 费用计算（买入费、卖出费）
  - 选股逻辑（基于5日均线）
  - 买卖执行（下单、成交处理）

### run_strategy.py
启动脚本，用于：
- 导入配置和策略
- 初始化策略对象
- 运行策略

## 使用方法

### 基本使用

```python
from ma5_strategy import MA5Strategy
from config import QMT_PATH, ACCOUNT_ID, CAPITAL_PER_STOCK

# 定义股票池
stock_pool = ["000001.SZ", "000002.SZ", "600000.SH", ...]

# 创建策略
strategy = MA5Strategy(QMT_PATH, ACCOUNT_ID, CAPITAL_PER_STOCK)

# 运行策略
strategy.run(stock_pool)
```

### 命令行运行

```bash
python run_strategy.py
```

## 策略逻辑

### 选股条件
- 股价 > 5日均线

### 买入条件
- 股票被选中（满足选股条件）
- 未持仓过该股票
- 资金充足

### 卖出条件
1. 股价 < 5日均线（跌破均线）
2. 收益 > 6%（止盈）
3. 亏损 > 5%（止损）

## 费用计算

### 买入费用
- 规则：每1万元收取3元，不满1万仍收3元
- 例：买入15,000元 → 手续费 = 3 + 3 = 6元

### 卖出费用
- 印花税：每1万5元，不满1万仍收5元
- 交易费：每1万3元，不满1万仍收3元
- 例：卖出15,000元 → 手续费 = (5+3) + (5+3) = 16元

5. 建议在非交易时间先测试连接
