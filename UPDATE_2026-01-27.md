## 💰 费用计算详解

### 新增功能 - 精确费用计算

#### 买入费用规则
```python
def calculate_buy_fee(self, amount):
    """
    每1万元收取3元
    不满1万的部分仍然收取3元
    
    例子：
    - 买入10,000元  → 费用 = 3元
    - 买入15,000元  → 费用 = 3 + 3 = 6元
    - 买入50,000元  → 费用 = 3 × 5 = 15元
    - 买入55,000元  → 费用 = 3 × 5 + 3 = 18元
    """
    whole_parts = int(amount / 10000)
    remainder = amount % 10000
    fee = whole_parts * 3
    if remainder > 0:
        fee += 3
    return fee
```

#### 卖出费用规则
```python
def calculate_sell_fee(self, amount):
    """
    印花税：每1万5元，不满1万仍收5元
    交易费：每1万3元，不满1万仍收3元
    总费用 = 印花税 + 交易费
    
    例子：
    - 卖出10,000元  → 费用 = (5 + 3) = 8元
    - 卖出15,000元  → 费用 = (5 + 3) + (5 + 3) = 16元
    - 卖出50,000元  → 费用 = (5 + 3) × 5 = 40元
    """
    whole_parts = int(amount / 10000)
    remainder = amount % 10000
    
    stamp_tax = whole_parts * 5
    trade_fee = whole_parts * 3
    
    if remainder > 0:
        stamp_tax += 5
        trade_fee += 3
    
    return stamp_tax + trade_fee
```

### 在回测中的应用

#### 买入时
```python
cost = price * quantity
buy_fee = self.calculate_buy_fee(cost)
total_cost = cost + buy_fee
self.capital -= total_cost
```

#### 卖出时
```python
proceeds = price * quantity
sell_fee = self.calculate_sell_fee(proceeds)
net_proceeds = proceeds - sell_fee
self.capital += net_proceeds
```

## 📊 策略逻辑（保持不变）

### 选股条件
- 股价 > 5日均线
- （可选）前10日涨幅 > 12%

### 买入条件
- 股票被选中
- 未持仓该股票
- 资金充足（包括手续费）

### 卖出条件
1. **跌破均线** - 股价 < 5日均线
2. **止盈** - 收益率 > 6%
3. **止损** - 亏损 > 5%
